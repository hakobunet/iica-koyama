## 1. React とは

**React = UI を「コンポーネント（部品）」で作る JavaScript ライブラリ**

```
従来の HTML + JavaScript：
  <div id="app"></div>
  document.getElementById("app").innerHTML = "...";
  → ページが増えると管理が大変

React：
  <Button label="送信" />  ← 再利用できる部品
  <Card title="タイトル" body="内容" />
  → 部品を組み合わせてページを作る
```

## 2. JSX — JavaScript の中に HTML を書く

> **どんな時に使う？** HTML だけでは変数の値を直接書けない。JSX を使うと `{変数名}` でリアルタイムに値を埋め込める。ユーザー名・日付・計算結果など「動的な値」を表示したい場面で使う。  
> **確認ポイント：** `namae` を自分の名前に変えて保存 → 画面が自動更新されるか確認。`{1 + 1}` が `2` と表示されているか確認。

```jsx
// ▼ src/App.jsx に書いてみよう（ファイルの中身を全部消して以下に書き換える）

// JSX = JavaScript + XML（HTMLのような記法）

// 通常の関数（コンポーネント）
function Hello() {
  // JSX を return する
  return (
    <div>
      <h1>こんにちは！</h1>
      <p>React を始めよう</p>
    </div>
  );
}

// JavaScript の式は {} で埋め込む
function Greeting() {
  const namae = "田中太郎";
  const today = new Date().toLocaleDateString("ja-JP");
  
  return (
    <div>
      <h1>こんにちは、{namae}さん！</h1>
      <p>今日は {today} です</p>
      <p>1 + 1 = {1 + 1}</p>
    </div>
  );
}

export default Greeting;
```

**JSX の注意点：**

> **どんな時に使う？** JSX を書いていてエラーが出た時、この3つのルールが原因のことが多い。赤いエラーが出たらここに戻って照合しよう。

```jsx
// ▼ 書き方の参考メモ（これ単体では動かない・読んで覚えよう）

// ① クラスは className（class はJSの予約語）
<div className="container">

// ② 全ての要素は閉じる
<input type="text" />  // 自己終了タグ

// ③ 複数要素は1つの親で囲む
return (
  <div>
    <h1>タイトル</h1>
    <p>内容</p>
  </div>
);
// または Fragment（<></>) を使う
return (
  <>
    <h1>タイトル</h1>
    <p>内容</p>
  </>
);
```

## 3. props — コンポーネントにデータを渡す

> **どんな時に使う？** 同じデザインで「データだけ違う」カードや行を複数表示したい時。コンポーネントを1つ書けば、渡すデータを変えるだけで何個でも使い回せる。  
> **確認ポイント：** `passed={false}` の StudentCard の背景色がオレンジ系になるか確認。4人目の `<StudentCard>` を自分で追加してみよう。

```jsx
// ▼ src/App.jsx に書いてみよう（ファイルの中身を全部消して以下に書き換える）

// props = コンポーネントへの「引数」

// コンポーネント定義（propsを受け取る）
function StudentCard({ name, score, passed }) {
  return (
    <div style={{
      border: "1px solid #ddd",
      padding: "16px",
      margin: "8px",
      backgroundColor: passed ? "#e3f2fd" : "#fbe9e7"
    }}>
      <h3>{name}</h3>
      <p>点数: {score}点</p>
      <span>{passed ? "✅ 合格" : "❌ 不合格"}</span>
    </div>
  );
}

// 使い方（親コンポーネント）
function App() {
  return (
    <div>
      <StudentCard name="田中太郎" score={85} passed={true} />
      <StudentCard name="鈴木花子" score={45} passed={false} />
      <StudentCard name="山田次郎" score={91} passed={true} />
    </div>
  );
}

export default App;
```

## 4. useState — 状態の管理

> **どんな時に使う？** ボタンクリック・入力・選択など「ユーザー操作で変わる値」をその場で画面に反映したい時。`count` のような「変化する値」は必ず `useState` で管理する。普通の変数（`let count = 0`）では画面は更新されない。  
> **確認ポイント：** `+1` を5回押して数字が増えるか確認。`リセット` で 0 に戻るか確認。ページがリロードされないことも確認。

```jsx
// ▼ src/App.jsx に書いてみよう（ファイルの中身を全部消して以下に書き換える）
import { useState } from "react";

function Counter() {
  // useState(初期値) → [現在の値, 値を変える関数]
  const [count, setCount] = useState(0);
  
  return (
    <div>
      <p>カウント: {count}</p>
      
      {/* ボタンを押すたびに count が増える */}
      <button onClick={() => setCount(count + 1)}>+1</button>
      <button onClick={() => setCount(count - 1)}>-1</button>
      <button onClick={() => setCount(0)}>リセット</button>
    </div>
  );
}

export default Counter;
```

## 5. リストのレンダリング

> **どんな時に使う？** 商品一覧・生徒リスト・コメント欄など「配列データを繰り返し表示」したい時に必ず使うパターン。データが増えれば表示も自動で増える。  
> **確認ポイント：** `students` 配列に4人目を追加してみよう → `<li>` が自動で増えるか確認。`key={student.id}` を消してブラウザのコンソール（F12）に警告が出ることも確認してみよう。

```jsx
// ▼ src/App.jsx に書いてみよう（ファイルの中身を全部消して以下に書き換える）
function StudentList() {
  const students = [
    { id: 1, name: "田中太郎", score: 85 },
    { id: 2, name: "鈴木花子", score: 72 },
    { id: 3, name: "山田次郎", score: 91 },
  ];
  
  return (
    <ul>
      {/* 配列の map で要素を生成 */}
      {/* key は Reactが差分を把握するために必要（ループ内で必ずつける） */}
      {students.map((student) => (
        <li key={student.id}>
          {student.name} - {student.score}点
        </li>
      ))}
    </ul>
  );
}

export default StudentList;
```

## 6. フォームとイベント

> **どんな時に使う？** 検索・ログイン・データ入力など「ユーザーの入力値を取得して処理したい」時。`onChange` で入力のたびに state を更新し、`onSubmit` でボタンを押した時の処理を行う。  
> **確認ポイント：** 何か入力して「検索」を押すと結果テキストが表示されるか確認。`e.preventDefault()` をコメントアウト（`//` を付ける）して送信時にページがリロードされることも確認してみよう。

```jsx
// ▼ src/App.jsx に書いてみよう（ファイルの中身を全部消して以下に書き換える）
import { useState } from "react";

function SearchForm() {
  const [query, setQuery] = useState("");
  const [result, setResult] = useState(null);
  
  function handleSubmit(e) {
    e.preventDefault();  // ページのリロードを防ぐ
    setResult(`"${query}" を検索しました`);
  }
  
  return (
    <form onSubmit={handleSubmit}>
      <input
        type="text"
        value={query}
        onChange={(e) => setQuery(e.target.value)}
        // onChange = 入力が変わるたびに setQuery で state を更新
        placeholder="検索キーワード"
      />
      <button type="submit">検索</button>
      {result && <p>{result}</p>}
    </form>
  );
}

export default SearchForm;
```

### 7-3. Step 1｜LoginForm.jsx を作る

`src/components/LoginForm.jsx` を新規作成して、以下をコピーして貼り付けます。

```jsx
import { useState } from "react";

// モック用の固定ログイン情報（本物のAPIの代わり）
const MOCK_EMAIL = "test@example.com";
const MOCK_PASSWORD = "password";
const MOCK_NAME = "テストユーザー";

// onLogin = ログイン成功時に App.jsx から渡してもらう関数
function LoginForm({ onLogin }) {
  const [email, setEmail] = useState("test@example.com");
  const [password, setPassword] = useState("password");
  const [error, setError] = useState("");

  function handleSubmit(e) {
    e.preventDefault(); // ページのリロードを防ぐ

    // 入力値がモック情報と一致すればログイン成功
    if (email === MOCK_EMAIL && password === MOCK_PASSWORD) {
      onLogin(MOCK_NAME); // 親コンポーネントにユーザー名を渡す
    } else {
      setError("メールアドレスまたはパスワードが違います");
    }
  }

  return (
    <div style={{
      maxWidth: "400px", margin: "80px auto",
      background: "white", padding: "32px",
      borderRadius: "8px", boxShadow: "0 2px 8px rgba(0,0,0,0.1)"
    }}>
      <h2 style={{ marginBottom: "24px", textAlign: "center" }}>ログイン</h2>

      <form onSubmit={handleSubmit}>
        <div style={{ marginBottom: "16px" }}>
          <label style={{ display: "block", marginBottom: "6px", fontWeight: "bold" }}>
            メールアドレス
          </label>
          <input
            type="email"
            value={email}
            onChange={(e) => setEmail(e.target.value)}
            style={{ width: "100%", padding: "10px", border: "1px solid #ccc", borderRadius: "6px", fontSize: "1em", boxSizing: "border-box" }}
          />
        </div>

        <div style={{ marginBottom: "16px" }}>
          <label style={{ display: "block", marginBottom: "6px", fontWeight: "bold" }}>
            パスワード
          </label>
          <input
            type="password"
            value={password}
            onChange={(e) => setPassword(e.target.value)}
            style={{ width: "100%", padding: "10px", border: "1px solid #ccc", borderRadius: "6px", fontSize: "1em", boxSizing: "border-box" }}
          />
        </div>

        {/* error が空でない場合だけエラーを表示 */}
        {error && (
          <p style={{ color: "#D32F2F", fontSize: "0.9em", marginBottom: "12px" }}>
            {error}
          </p>
        )}

        <button
          type="submit"
          style={{ width: "100%", padding: "12px", background: "#1976D2", color: "white", border: "none", borderRadius: "6px", fontSize: "1em", fontWeight: "bold", cursor: "pointer" }}
        >
          ログイン
        </button>
      </form>

      <p style={{ marginTop: "16px", fontSize: "0.85em", color: "#888", textAlign: "center" }}>
        ※ Mock用: test@example.com / password
      </p>
    </div>
  );
}

export default LoginForm;
```

```jsx
// StudentList：生徒一覧テーブル全体
// students = 生徒データの配列（親から渡ってくる）
// onDelete = 削除ボタンを押したときに呼ぶ関数（親から渡ってくる）
function StudentList({ students, onDelete }) {
  return (

                key={student.id}
                student={student}
                onDelete={onDelete}
              />

// StudentRow：テーブルの1行分
function StudentRow({ student, onDelete }) {
  function handleDelete() {
    // 確認ダイアログを表示してから削除
    if (window.confirm(`${student.name} を削除しますか？`)) {
      onDelete(student.id); // 親に「このIDを削除して」と伝える
    }
  }

  return (
    <tr style={{ borderBottom: "1px solid #eee" }}>
      <td style={tdStyle}>{student.id}</td>
      <td style={tdStyle}>{student.name}</td>
      <td style={tdStyle}>{student.email}</td>
      <td style={tdStyle}>{student.score}</td>

        <button
          onClick={handleDelete}
          style={{ background: "#D32F2F", color: "white", border: "none", borderRadius: "4px", padding: "4px 12px", cursor: "pointer", fontSize: "0.85em" }}
        >
          削除
        </button>
      </td>
    </tr>
  );
}

// スタイル定数（繰り返し使う style をまとめる）

const tdStyle = { padding: "12px 16px", fontSize: "0.95em" };
```

### 7-5. Step 3｜AddStudentForm.jsx を作る

`src/components/AddStudentForm.jsx` を新規作成して、以下をコピーして貼り付けます。

```jsx
import { useState } from "react";

// onAdd = 追加ボタンを押したときに呼ぶ関数（親から渡ってくる）
function AddStudentForm({ onAdd }) {
  const [name, setName] = useState("");
  const [email, setEmail] = useState("");
  const [score, setScore] = useState("");
  const [error, setError] = useState("");

  function handleSubmit(e) {
    e.preventDefault();
    setError(""); // エラーをリセット

    // バリデーション（入力チェック）
    if (!name || !email || !score) {
      setError("すべての項目を入力してください");
      return;
    }
    if (Number(score) < 0 || Number(score) > 100) {
      setError("点数は0〜100の数値を入力してください");
      return;
    }

    // 親コンポーネントに新しい生徒データを渡す
    onAdd({ name, email, score: Number(score) });

    // フォームをリセット
    setName("");
    setEmail("");
    setScore("");
  }

  return (
    <div style={{ background: "white", borderRadius: "8px", padding: "24px", boxShadow: "0 2px 8px rgba(0,0,0,0.05)", marginBottom: "24px" }}>
      <h2 style={{ marginBottom: "16px", fontSize: "1.1em" }}>新規追加</h2>

      <form onSubmit={handleSubmit} style={{ display: "flex", gap: "12px", alignItems: "flex-end", flexWrap: "wrap" }}>
        <div>
          <label style={labelStyle}>名前</label>
          <input
            type="text"
            value={name}
            onChange={(e) => setName(e.target.value)}
            placeholder="田中太郎"
            style={inputStyle}
          />
        </div>
        <div>
          <label style={labelStyle}>メールアドレス</label>
          <input
            type="email"
            value={email}
            onChange={(e) => setEmail(e.target.value)}
            placeholder="tanaka@example.com"
            style={inputStyle}
          />
        </div>
        <div>
          <label style={labelStyle}>点数</label>
          <input
            type="number"
            value={score}
            onChange={(e) => setScore(e.target.value)}
            placeholder="85"
            min="0"
            max="100"
            style={{ ...inputStyle, width: "80px" }}
          />
        </div>
```

### 7-6. Step 4｜App.jsx で全部つなぐ

`src/App.jsx` を開いて、**中身を全部消して**以下に書き換えます。

```jsx
import { useState } from "react";
import LoginForm from "./components/LoginForm";
import StudentList from "./components/StudentList";
import AddStudentForm from "./components/AddStudentForm";

// 最初から表示するモックデータ
const INITIAL_STUDENTS = [
  { id: 1, name: "田中太郎", email: "tanaka@example.com", score: 85 },
  { id: 2, name: "山田花子", email: "yamada@example.com", score: 72 },
  { id: 3, name: "山田次郎", email: "jiro@example.com", score: 91 },
];

function App() {
  // ログイン状態を管理
  const [isLoggedIn, setIsLoggedIn] = useState(false);
  const [userName, setUserName] = useState("");

  // 生徒リストを管理
  const [students, setStudents] = useState(INITIAL_STUDENTS);
  const [nextId, setNextId] = useState(4); // 次に追加する生徒のID

  // ログイン成功時：LoginForm から呼ばれる
  function handleLogin(name) {
    setIsLoggedIn(true);
    setUserName(name);
  }

  // ログアウト時
  function handleLogout() {
    setIsLoggedIn(false);
    setUserName("");
  }

  // 生徒を追加：AddStudentForm から呼ばれる
  function handleAdd(newStudent) {
    setStudents([...students, { ...newStudent, id: nextId }]);
    setNextId(nextId + 1);
  }

  // 生徒を削除：StudentList から呼ばれる
  function handleDelete(id) {
    setStudents(students.filter((s) => s.id !== id));
  }

  return (
    <div style={{ background: "#f4f6f8", minHeight: "100vh" }}>

      {/* ヘッダー */}
      <header style={{
        background: "#1565C0", color: "white",
        padding: "16px 32px",
        display: "flex", alignItems: "center"
      }}>
        <h1 style={{ fontSize: "1.2em" }}>生徒管理アプリ（React Mock版）</h1>

        {/* ログイン中だけヘッダー右側にユーザー名とログアウトボタンを表示 */}
        {isLoggedIn && (
          <div style={{ marginLeft: "auto", display: "flex", alignItems: "center", gap: "12px" }}>
            <span style={{ fontSize: "0.9em" }}>● ログイン中：{userName}</span>
            <button
              onClick={handleLogout}
              style={{ background: "#D32F2F", color: "white", border: "none", borderRadius: "6px", padding: "6px 14px", cursor: "pointer" }}
            >
              ログアウト
            </button>
          </div>
        )}
      </header>

      {/* メインコンテンツ */}
      <main style={{ padding: "32px", maxWidth: "900px", margin: "0 auto" }}>
        {/* isLoggedIn が true なら生徒管理画面、false ならログイン画面 */}
        {isLoggedIn ? (
          <>
            <AddStudentForm onAdd={handleAdd} />
            <StudentList students={students} onDelete={handleDelete} />
          </>
        ) : (
          <LoginForm onLogin={handleLogin} />
        )}
      </main>

    </div>
  );
}

export default App;
```

### 7-8. コードの流れを整理する

```
App.jsx（司令塔）
│
│  isLoggedIn が false のとき
├─── <LoginForm onLogin={handleLogin} />
│         ↓ ログイン成功
│         onLogin("テストユーザー") を呼ぶ
│         → App の isLoggedIn が true になる
│
│  isLoggedIn が true のとき
├─── <AddStudentForm onAdd={handleAdd} />
│         ↓ 追加ボタンを押す
│         onAdd({name, email, score}) を呼ぶ
│         → App の students 配列に追加される
│
└─── <StudentList students={students} onDelete={handleDelete} />
          ↓ 削除ボタンを押す
          onDelete(id) を呼ぶ
          → App の students 配列から該当IDが消える
```

> **ポイント：** データ（状態）は `App.jsx` が一括管理して、  
> 子コンポーネントには「データ」と「操作する関数」を props で渡すのが React の基本パターンです。

### 7-9. Next.js との違い（次回の予習）

| | React（今回） | Next.js（次回） |
|--|--|--|
| データの取得 | コード内の Mock 配列 | `fetch('/api/students')` で Laravel API から取得 |
| ログイン | 固定値と照合 | `POST /api/login` でトークン取得 |
| 削除 | 配列を filter で更新 | `DELETE /api/students/{id}` を呼んで再取得 |
| 画面の見た目 | **全く同じ** | **全く同じ** |

次回は **今回作った画面をそのまま使って**、データの取得部分だけを API に繋ぎ替えます。
