## 演習1：セルに情報を書き込む

### 問題

> **目安時間：10〜15分**

GASを使って、自分の自己紹介をスプレッドシートに書き込む関数を作りましょう。

**やること：**

1. `jikoshokaSheet` という名前の関数を作る
2. 現在のシートを `sheet` という変数に取得する（資料の3章を参考に）
3. 下の表のとおりに、各セルに値を書き込む

| セル | 書き込む内容 |
|------|------------|
| A1 | `"自己紹介"` という文字列 |
| B1 | **自分の名前**（例：`"田中太郎"`） |
| A2 | `"年齢"` という文字列 |
| B2 | **自分の年齢**（数値。クォートなし） |

4. **さらに**、A3〜B4 を使って、好きな情報（趣味・出身地など）を自分で追加してみよう

> **ヒント：** セルへの書き込みは `getRange("A1").setValue(値)` を使います（資料3章）

**実行してシートに書き込まれたことを確認しましょう**

### 答え

```javascript
function jikoshokaSheet() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  
  // A1 に「自己紹介」と書く
  sheet.getRange("A1").setValue("自己紹介");
  
  // B1 に自分の名前を書く（← 変えてください）
  sheet.getRange("B1").setValue("田中太郎");
  
  // A2 に「年齢」、B2 に年齢の数値を書く
  sheet.getRange("A2").setValue("年齢");
  sheet.getRange("B2").setValue(18);
  
  // A3、B3、A4、B4 も自分で追加してみましょう
}
```

## 演習2：成績表を一気に書き込む

### 問題

> **目安時間：20〜30分**

1セルずつ書くのは大変です。今度は **2次元配列と `setValues()`** を使って、成績表を一気に書き込む関数を作りましょう。

**やること：**

1. `seisekiTable` という名前の関数を作る
2. 最初にシートを **すべてクリア** する（`clearContents()` を使う）
3. 1行目に `"番号"`, `"名前"`, `"点数"` のヘッダーを書き込む
   - **2次元配列** `[["番号", "名前", "点数"]]` を `setValues()` で書き込む
   - `getRange(1, 1, 1, 3)` で「1行目・1列目から、1行×3列」を指定する（資料5章）
4. 2行目以降に、以下の5人分のデータを書き込む

| 番号 | 名前 | 点数 |
|------|------|------|
| 1 | 田中太郎 | 85 |
| 2 | 鈴木花子 | 72 |
| 3 | 山田次郎 | 91 |
| 4 | 佐藤三郎 | 68 |
| 5 | 伊藤美咲 | 78 |

   - データを2次元配列で用意し、`getRange(2, 1, データの行数, データの列数)` で書き込む
5. ヘッダー行（1行目）を **太字** にして **グレーの背景色**（`"#CCCCCC"`）にする（資料7章）

> **ヒント：** `data.length` で行数、`data[0].length` で列数を取得できます

### 答え

```javascript
function seisekiTable() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  
  // シートをクリアする（前のデータを消す）
  sheet.clearContents();
  
  // ヘッダー行
  const header = [["番号", "名前", "点数"]];
  sheet.getRange(1, 1, 1, 3).setValues(header);
  
  // データ行（5人分）
  const data = [
    [1, "田中太郎", 85],
    [2, "鈴木花子", 72],
    [3, "山田次郎", 91],
    [4, "佐藤三郎", 68],
    [5, "伊藤美咲", 78],
  ];
  
  // 2行目から書き込む
  sheet.getRange(2, 1, data.length, data[0].length).setValues(data);
  
  // ヘッダーを太字にする
  sheet.getRange(1, 1, 1, 3).setFontWeight("bold");
  sheet.getRange(1, 1, 1, 3).setBackground("#CCCCCC");
}
```

## 演習3：セルを読み取って加工する

### 問題

> **目安時間：30〜40分**

**演習2 を実行した後に取り組みましょう。**

演習2で作った成績表の点数を読み取り、D列に「合否」を自動判定して書き込む関数を作りましょう。

**やること：**

1. `readAndProcess` という名前の関数を作る

2. `getLastRow()` でデータが入っている最終行番号を取得する（資料6章）

3. D列（4列目）の1行目に、ヘッダー `"合否"` を書き込む
   - `getRange(1, 4)` で「1行目・4列目」を指定する

1. `for` ループで 2行目〜最終行まで繰り返す
   - 各行のC列（3列目）の点数を `getValue()` で読み取る
   - 点数が **60点以上なら `"合格"`**、**未満なら `"不合格"`** と判定する
     - 三項演算子 `score >= 60 ? "合格" : "不合格"` を使う
   - 判定結果をD列（4列目）に `setValue()` で書き込む

1. ループが終わったら `console.log("合否判定が完了しました！")` を出力する

> **ヒント：** `getRange(行番号, 列番号)` で行・列を数値で指定できます（資料4章）

**実行後、D列に合否が表示されたか確認しましょう**

### 答え

```javascript
function readAndProcess() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  
  // 演習2 を実行した後に、このコードを実行してください
  
  // 2行目から最終行までのデータを読み取る
  const lastRow = sheet.getLastRow();
  
  // D列（4列目）に「合否」を書き込む
  // ヘッダー
  sheet.getRange(1, 4).setValue("合否");
  
  for (let row = 2; row <= lastRow; row++) {
    // C列（3列目）の点数を読み取る
    const score = sheet.getRange(row, 3).getValue();
    
    // 60点以上なら合格、未満なら不合格
    const result = score >= 60 ? "合格" : "不合格";
    
    // D列に書き込む
    sheet.getRange(row, 4).setValue(result);
  }
  
  console.log("合否判定が完了しました！");
}
```

## 演習4：書式を整えよう

### 問題

> **目安時間：15〜20分**

**演習3 を実行した後に取り組みましょう。**

演習3で書き込んだ「合否」列を読み取り、合格・不合格の行に **背景色** をつけて、結果が一目でわかる成績表に仕上げましょう。

**やること：**

1. `colorFormat` という名前の関数を作る
2. `getLastRow()` で最終行番号を取得する
3. `for` ループで 2行目〜最終行まで繰り返す
   - D列（4列目）の値を `getValue()` で読み取る
   - 値が `"合格"` なら、その行（1〜4列目）の背景色を **薄い青** `"#CCE5FF"` にする
   - 値が `"不合格"` なら、その行の背景色を **薄いオレンジ** `"#FFE5CC"` にする
   - 1行分の横幅を指定するには `getRange(行番号, 1, 1, 4)` を使う（「行番号行・1列目から、1行×4列」）

> **確認ポイント：** 実行後、合格の行が青、不合格の行がオレンジになっていれば成功です

### 答え

```javascript
function colorFormat() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  const lastRow = sheet.getLastRow();
  
  for (let row = 2; row <= lastRow; row++) {
    const result = sheet.getRange(row, 4).getValue();
    
    if (result === "合格") {
      // 合格行を薄い青にする
      sheet.getRange(row, 1, 1, 4).setBackground("#CCE5FF");
    } else {
      // 不合格行を薄いオレンジにする
      sheet.getRange(row, 1, 1, 4).setBackground("#FFE5CC");
    }
  }
}
```
