## 1. API とは？

**API（Application Programming Interface）= アプリ同士が会話するための窓口**

### 身近な例で考える

```
レストランの例：
─────────────────────────────────────────
  あなた（客）    ←→    ウェイター    ←→    厨房
  （ブラウザ）          （API）          （サーバー）
─────────────────────────────────────────
① 「カレーをください」（リクエスト）
② ウェイターが厨房に伝える
③ 厨房が料理する
④ 「はい、カレーです」（レスポンス）
```

あなたは厨房（サーバー）の内部構造を知らなくていい。  
ウェイター（API）を通じて注文するだけ。

## 2. HTTP リクエストとレスポンス

```
クライアント（GAS）                  サーバー（天気サービスなど）
        │                                    │
        │── GET /weather?city=Tokyo ────────→│  リクエスト
        │                                    │（処理中...）
        │← 200 OK { "temp": 25, ... } ──────│  レスポンス
        │                                    │
```

**リクエストの種類：**
| メソッド | 意味 | 例 |
|---------|------|-----|
| GET | データを取得したい | 天気を取ってくる |
| POST | データを送信・作成したい | フォームを送信する |
| PUT | データを更新したい | プロフィールを変更する |
| DELETE | データを削除したい | 投稿を削除する |

**レスポンスの状態コード：**
| コード | 意味 |
|--------|------|
| 200 | OK（成功） |
| 404 | Not Found（見つからない） |
| 500 | Internal Server Error（サーバーエラー） |

## 3. JSON — データのやり取り形式

APIのデータは主に **JSON（JavaScript Object Notation）** 形式で来ます。

```json
{
  "city": "Tokyo",
  "weather": "晴れ",
  "temperature": 25.5,
  "humidity": 60,
  "forecast": [
    { "date": "2024-01-15", "temp": 24 },
    { "date": "2024-01-16", "temp": 22 },
    { "date": "2024-01-17", "temp": 27 }
  ]
}
```

JavaScript のオブジェクトとほぼ同じ書き方。コマ08で学んだ知識がそのまま使えます！

## 4. GAS で外部 API を呼び出す

```javascript
function callApi() {
  // UrlFetchApp.fetch(URL) = 指定したURLにアクセスしてデータを取得
  const url = "https://api.example.com/data";
  
  const response = UrlFetchApp.fetch(url);
  
  // レスポンスの内容を文字列で取得
  const text = response.getContentText();
  console.log(text);  // JSON 文字列が表示される
  
  // JSON 文字列をオブジェクトに変換（コマ08で学んだ！）
  const data = JSON.parse(text);
  console.log(data.temperature);  // → 25.5
}
```

## 5. 公開APIを実際に使ってみる

### 例：Open-Meteo（無料の天気API、APIキー不要）

```javascript
function getWeather() {
  // 東京の緯度・経度を指定して天気データを取得
  const url = "https://api.open-meteo.com/v1/forecast?latitude=35.6762&longitude=139.6503&current_weather=true";
  
  // APIにリクエストを送る
  const response = UrlFetchApp.fetch(url);
  
  // レスポンスをJSON形式で取得
  const json = JSON.parse(response.getContentText());
  
  // current_weather の中に現在の気象データが入っている
  const weather = json.current_weather;
  
  console.log("気温: " + weather.temperature + "℃");
  console.log("風速: " + weather.windspeed + " km/h");
  
  // 天気コードを日本語に変換（一部）
  const weatherCodes = {
    0: "快晴",
    1: "晴れ",
    2: "一部曇り",
    3: "曇り",
    61: "弱い雨",
    63: "適度な雨",
    65: "強い雨",
  };
  
  const weatherText = weatherCodes[weather.weathercode] || "不明";
  console.log("天気: " + weatherText);
}
```

## 6. APIのレスポンスをシートに書き込む

```javascript
function weatherToSheet() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  
  // APIから天気データを取得
  const url = "https://api.open-meteo.com/v1/forecast?latitude=35.6762&longitude=139.6503&current_weather=true";
  const response = UrlFetchApp.fetch(url);
  const data = JSON.parse(response.getContentText());
  const weather = data.current_weather;
  
  // 現在時刻を取得
  const now = new Date();
  const timeStr = Utilities.formatDate(now, "Asia/Tokyo", "yyyy/MM/dd HH:mm");
  
  // 最終行の次に書き込む
  const nextRow = sheet.getLastRow() + 1;
  sheet.getRange(nextRow, 1).setValue(timeStr);
  sheet.getRange(nextRow, 2).setValue(weather.temperature);
  sheet.getRange(nextRow, 3).setValue(weather.windspeed);
  
  console.log("天気データを記録しました！");
}
```

## まとめ

| 用語                       | 意味                |
| ------------------------ | ----------------- |
| API                      | アプリ同士が会話する窓口      |
| リクエスト                    | クライアントからサーバーへの要求  |
| レスポンス                    | サーバーからクライアントへの返答  |
| JSON                     | データのやり取りに使う形式     |
| `UrlFetchApp.fetch(url)` | GAS で URL にアクセスする |
| `JSON.parse(str)`        | JSON文字列をオブジェクトに変換 |
