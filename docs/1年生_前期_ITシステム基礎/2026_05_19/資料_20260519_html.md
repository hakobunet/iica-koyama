# HTML基礎 — タグ・構造・フォーム

## 今日のゴール
- HTMLの基本構造を書ける
- よく使うタグを使いこなせる
- フォームを作れる
- GAS の HTMLサービスで Web ページを表示できる

---

## 1. HTMLとは

**HTML（HyperText Markup Language）= Webページの「骨格」を作る言語**

```
Webページの構成要素：
├── HTML → 構造・内容（骨・肉）
├── CSS  → 見た目（服・化粧）
└── JS   → 動き（筋肉）
```

---

## 2. 基本構造

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <!-- head内はブラウザへの設定情報。画面には表示されない -->
  <meta charset="UTF-8">
  <title>ページタイトル</title>  <!-- タブに表示される -->
</head>
<body>
  <!-- body内に書いたものが画面に表示される -->
  <h1>見出し1</h1>
  <p>段落テキスト</p>
</body>
</html>
```

---

## 3. よく使うタグ

### 見出し・段落

```html
<h1>大見出し（一番大きい）</h1>
<h2>中見出し</h2>
<h3>小見出し</h3>
<p>段落テキスト。改行は自動。</p>
<br>  <!-- 強制改行 -->
<hr>  <!-- 水平線 -->
```

### リンクと画像

```html
<!-- リンク -->
<a href="https://google.com">Google へ</a>
<a href="page2.html">次のページ</a>          <!-- 同じサイト内 -->
<a href="https://google.com" target="_blank">新しいタブで開く</a>

<!-- 画像 -->
<img src="photo.jpg" alt="写真の説明">
<!-- alt = 画像が表示できないときの代替テキスト -->
```

### リスト

```html
<!-- 番号なしリスト（ul = unordered list）-->
<ul>
  <li>りんご</li>
  <li>みかん</li>
  <li>ぶどう</li>
</ul>

<!-- 番号ありリスト（ol = ordered list）-->
<ol>
  <li>手を洗う</li>
  <li>野菜を切る</li>
  <li>炒める</li>
</ol>
```

### テーブル（表）

```html
<table border="1">
  <thead>
    <tr>
      <th>名前</th>   <!-- th = ヘッダーセル（太字・中央寄せ） -->
      <th>点数</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>田中太郎</td>   <!-- td = データセル -->
      <td>85</td>
    </tr>
    <tr>
      <td>鈴木花子</td>
      <td>72</td>
    </tr>
  </tbody>
</table>
```

### div と span（グループ分け）

```html
<!-- div = ブロック要素のグループ（縦に並ぶ） -->
<div class="box">
  <h2>タイトル</h2>
  <p>内容</p>
</div>

<!-- span = インライン要素のグループ（横に並ぶ） -->
<p>テキストの<span style="color: red;">一部だけ</span>を変える</p>
```

## 4. フォーム

ユーザーから入力を受け付ける部品です。

```html
<form action="" method="post">
  <!-- テキスト入力 -->
  <label for="namae">名前：</label>
  <input type="text" id="namae" name="namae" placeholder="名前を入力">
  
  <br>
  
  <!-- メール入力 -->
  <label for="email">メール：</label>
  <input type="email" id="email" name="email">
  
  <br>
  
  <!-- 数値入力 -->
  <label for="age">年齢：</label>
  <input type="number" id="age" name="age" min="0" max="150">
  
  <br>
  
  <!-- セレクトボックス -->
  <label for="grade">学年：</label>
  <select id="grade" name="grade">
    <option value="1">1年生</option>
    <option value="2">2年生</option>
    <option value="3">3年生</option>
  </select>
  
  <br>
  
  <!-- テキストエリア（複数行） -->
  <label for="comment">コメント：</label>
  <textarea id="comment" name="comment" rows="4" cols="40"></textarea>
  
  <br>
  
  <!-- 送信ボタン -->
  <button type="submit">送信する</button>
</form>
```

## 5. GAS の HTMLサービスで Web ページを表示する

GASを使ってWebページを公開できます！

**ファイル構成（GASエディタで作成）：**
- `コード.gs`（GASのメイン）
- `index.html`（表示するHTMLファイル）

**コード.gs：**
```javascript
// Webアプリとして公開するための関数
// URLにアクセスされたときに呼ばれる
function doGet() {
  // index.html を読み込んで表示する
  return HtmlService.createHtmlOutputFromFile("index")
    .setTitle("私のWebページ");
}
```

**index.html：**
```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>GASのWebページ</title>
</head>
<body>
  <h1>こんにちは！GAS HTML サービス</h1>
  <p>これはGASで表示したWebページです。</p>
  
  <ul>
    <li>スプレッドシートのデータを表示できる</li>
    <li>フォームの送信を受け取れる</li>
    <li>インターネットで公開できる</li>
  </ul>
</body>
</html>
```

**Webアプリとして公開する方法：**
1. 「デプロイ」→「新しいデプロイ」
2. 種類：「Webアプリ」を選択
3. アクセスできるユーザー：「全員」
4. 「デプロイ」→ URLが発行される！

## まとめ

| タグ | 用途 |
|------|------|
| `<h1>`〜`<h6>` | 見出し |
| `<p>` | 段落 |
| `<a href="">` | リンク |
| `<img src="">` | 画像 |
| `<ul>` / `<ol>` / `<li>` | リスト |
| `<table>` / `<tr>` / `<td>` | 表 |
| `<form>` | フォーム（入力受付） |
| `<input>` / `<button>` | 入力部品・ボタン |
| `<div>` / `<span>` | グループ分け |
