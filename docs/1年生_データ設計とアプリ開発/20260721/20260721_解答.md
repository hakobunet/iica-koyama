# コマ07 演習｜解答例

`students` テーブルのデータ（コマ07資料参照）をもとにした解答例です。

---

## 問題1（SELECT基礎）

### 答え

```sql
-- ① name と class だけ取得
SELECT name, class FROM students;

-- ② name AS 氏名, score AS 点数 で取得
SELECT name AS 氏名, score AS 点数 FROM students;

-- ③ 全カラム・全行を取得
SELECT * FROM students;
```

---

## 問題2（WHERE）

### 答え

```sql
-- ① class が 'B' の生徒
SELECT * FROM students WHERE class = 'B';

-- ② age が18歳以上、かつ score が80点以上
SELECT * FROM students WHERE age >= 18 AND score >= 80;

-- ③ class が 'A' または 'C'
SELECT * FROM students WHERE class IN ('A', 'C');

-- ④ name に「郎」が含まれる
SELECT * FROM students WHERE name LIKE '%郎%';
```

**結果（参考）：**
- ①：山田次郎(91)、佐藤三郎(58)、吉田健太(88)
- ②：田中太郎(85)、山田次郎(91)、渡辺一郎(95)、吉田健太(88)
- ③：田中太郎・鈴木花子・伊藤美咲・山本麻衣（A）、渡辺一郎・加藤幸子・中村龍一（C）
- ④：田中太郎・山田次郎・佐藤三郎・渡辺一郎（4名）

## 問題3（ORDER BY / LIMIT）

### 答え

```sql
-- ① score の降順で全員
SELECT * FROM students ORDER BY score DESC;

-- ② 上位3名だけ
SELECT * FROM students ORDER BY score DESC LIMIT 3;

-- ③ class 昇順、同じ class 内は score 降順
SELECT * FROM students ORDER BY class ASC, score DESC;
```

**結果（参考）：**

②
```
渡辺一郎 95点
山田次郎 91点
吉田健太 88点
```

③
```
[A] 田中太郎85, 伊藤美咲78, 鈴木花子72, 山本麻衣67
[B] 山田次郎91, 吉田健太88, 佐藤三郎58
[C] 渡辺一郎95, 中村龍一73, 加藤幸子45
```

## 問題4（集計関数）

### 答え

```sql
-- ① 人数
SELECT COUNT(*) FROM students;

-- ② 合計・平均・最高・最低をまとめて取得
SELECT
    SUM(score) AS 合計点,
    AVG(score) AS 平均点,
    MAX(score) AS 最高点,
    MIN(score) AS 最低点
FROM students;
```

**結果（参考）：**
- ①：10人
- ②：合計点 752 ／ 平均点 75.2 ／ 最高点 95 ／ 最低点 45

## 問題5（発展）

### 答え

```sql
-- ① 60点以上の生徒を対象に、class ごとの平均点・人数を平均点の高い順に
SELECT class, AVG(score) AS 平均点, COUNT(*) AS 人数
FROM students
WHERE score >= 60
GROUP BY class
ORDER BY 平均点 DESC;

-- ② class = 'A' の中で最高得点の生徒を1人だけ
SELECT * FROM students
WHERE class = 'A'
ORDER BY score DESC
LIMIT 1;
```

**結果（参考）：**

①（60点未満の佐藤三郎・加藤幸子は除外される）
```
B: 平均点 89.5, 人数 2
C: 平均点 84.0, 人数 2
A: 平均点 75.5, 人数 4
```

②
```
田中太郎 85点
```
