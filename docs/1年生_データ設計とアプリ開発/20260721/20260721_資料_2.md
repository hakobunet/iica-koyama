# コマ07｜SQL基礎① — SELECT / WHERE / ORDER BY

## 今日のゴール
- SELECT でデータを取り出せる
- WHERE で条件を指定して絞り込める
- ORDER BY で並び替えができる
- LIMIT で取得件数を制限できる

---

## 1. SELECT — データを取り出す

```sql
-- 全カラム・全行を取得（* = 全カラム）
SELECT * FROM students;

-- 特定のカラムだけ取得
SELECT name, score FROM students;

-- カラムに別名をつける（AS）
SELECT name AS 名前, score AS 点数 FROM students;
```

---

## 2. WHERE — 条件で絞り込む

```sql
-- score が 80 以上の生徒
SELECT * FROM students WHERE score >= 80;

-- class が 'A' の生徒
SELECT * FROM students WHERE class = 'A';

-- age が 18 かつ score が 70 以上
SELECT * FROM students WHERE age = 18 AND score >= 70;

-- class が 'A' または 'B'
SELECT * FROM students WHERE class = 'A' OR class = 'B';

-- class が 'A' または 'B'（IN を使うと短く書ける）
SELECT * FROM students WHERE class IN ('A', 'B');

-- score が 60 以上 90 未満（BETWEEN）
SELECT * FROM students WHERE score BETWEEN 60 AND 89;

-- name に "太" が含まれる（LIKE = 部分一致）
SELECT * FROM students WHERE name LIKE '%太%';
-- % = 何文字でもOK（ワイルドカード）

-- score が NULL でない（IS NOT NULL）
SELECT * FROM students WHERE score IS NOT NULL;
```

---

## 3. ORDER BY — 並び替え

```sql
-- score の昇順（低い順）
SELECT * FROM students ORDER BY score ASC;   -- ASC = ascending（昇順）

-- score の降順（高い順）
SELECT * FROM students ORDER BY score DESC;  -- DESC = descending（降順）

-- class で昇順、同じ class 内は score の降順
SELECT * FROM students ORDER BY class ASC, score DESC;
```

---

## 4. LIMIT — 取得件数を制限する

```sql
-- 上位5件だけ取得
SELECT * FROM students ORDER BY score DESC LIMIT 5;

-- 6件目から5件取得（ページネーションに使う）
-- LIMIT 件数 OFFSET 開始位置
SELECT * FROM students ORDER BY id ASC LIMIT 5 OFFSET 5;
```

---

## 5. 集計関数

```sql
-- 行数を数える
SELECT COUNT(*) FROM students;
-- → 10

-- score の合計・平均・最大・最小
SELECT
    COUNT(*)    AS 人数,
    SUM(score)  AS 合計点,
    AVG(score)  AS 平均点,
    MAX(score)  AS 最高点,
    MIN(score)  AS 最低点
FROM students;

-- GROUP BY：クラス別の平均点
SELECT class, AVG(score) AS 平均点, COUNT(*) AS 人数
FROM students
GROUP BY class
ORDER BY 平均点 DESC;
```

---

## 6. SELECT の実行順序（重要！）

```sql
SELECT name, score       -- ⑤ 取り出す列を選ぶ
FROM students            -- ① テーブルを選ぶ
WHERE score >= 70        -- ② 条件で絞り込む
GROUP BY class           -- ③ グループ化
HAVING AVG(score) >= 75  -- ④ グループに条件（WHERE の後）
ORDER BY score DESC      -- ⑥ 並び替え
LIMIT 5;                 -- ⑦ 件数制限
```

---

## まとめ

| SQL | 意味 |
|-----|------|
| `SELECT * FROM テーブル` | 全データを取得 |
| `WHERE 条件` | 条件で絞り込む |
| `AND` / `OR` | 複数条件の組み合わせ |
| `IN (値1, 値2)` | 複数の値のいずれかと一致 |
| `LIKE '%文字%'` | 部分一致検索 |
| `ORDER BY 列 ASC/DESC` | 並び替え |
| `LIMIT 数値` | 取得件数の制限 |
| `COUNT(*)` / `AVG()` / `SUM()` | 集計関数 |
| `GROUP BY 列` | グループ化 |
