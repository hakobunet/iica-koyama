## 1. 変数と型

```php
<?php
// 変数は必ず $ から始まる
$namae    = "田中太郎";   // 文字列（string）
$nenrei   = 20;           // 整数（int）
$身長     = 1.75;         // 小数（float）
$isStudent = true;        // 真偽値（bool）

// gettype() で型を確認（JS の typeof に相当）
echo gettype($namae);     // → string
echo gettype($nenrei);    // → integer

// 型変換
$str = "42";
$num = (int)$str;         // 文字列を整数に変換
echo $num + 1;            // → 43
```

## 2. 文字列操作

```php
<?php
$namae = "田中太郎";

echo mb_strlen($namae);   // → 4（文字数。mb_ 関数で日本語対応）

// 変数の埋め込み
echo "名前: {$namae}さん";

// よく使う文字列関数
echo strtolower("HELLO");              // → hello
echo strtoupper("hello");              // → HELLO
echo trim("  hello  ");                // → hello（前後の空白を削除）
echo str_replace("猫", "犬", "猫が好き"); // → 犬が好き
```

## 3. 配列

```php
<?php
$scores = [85, 72, 91, 68, 78];

echo $scores[0];          // → 85（0 始まり）
echo count($scores);      // → 5（JS の .length に相当）

$scores[] = 95;           // 末尾に追加（JS の push に相当）
sort($scores);            // 昇順に並び替え
rsort($scores);           // 降順に並び替え

// よく使う配列関数
$doubled  = array_map(fn($x) => $x * 2, $scores);       // JS の map
$big      = array_filter($scores, fn($x) => $x >= 80);  // JS の filter
$total    = array_sum($scores);                           // 合計
$average  = array_sum($scores) / count($scores);          // 平均
```

## 4. 連想配列（JS のオブジェクトに相当）

```php
<?php
// 連想配列 = キーと値のペア
$person = [
    "namae"  => "田中太郎",
    "nenrei" => 20,
    "email"  => "tanaka@example.com",
];

echo $person["namae"];   // → 田中太郎

$person["hobby"] = "プログラミング";  // キーを追加
$person["nenrei"] = 21;               // 値を変更

// foreach でキーと値を取り出す
foreach ($person as $key => $value) {
    echo "{$key}: {$value}\n";
}

// ========== 連想配列の配列（JS のオブジェクト配列） ==========
$students = [
    ["id" => 1, "name" => "田中太郎", "score" => 85],
    ["id" => 2, "name" => "鈴木花子", "score" => 72],
];

foreach ($students as $student) {
    echo "ID:{$student['id']} {$student['name']} {$student['score']}点\n";
}
```

## 5. 条件分岐

```php
<?php
$score = 72;

// if/elseif/else（PHP は elseif と書く）
if ($score >= 90) {
    echo "S評価";
} elseif ($score >= 80) {
    echo "A評価";
} elseif ($score >= 70) {
    echo "B評価";
} else {
    echo "D評価";
}

// 三項演算子（JS と同じ）
$result = ($score >= 60) ? "合格" : "不合格";

// match 式（PHP 8.0以降・switch の改良版）
$status = "active";
$label = match($status) {
    "active"   => "有効",
    "inactive" => "無効",
    default    => "不明",
};
echo $label;  // → 有効
```

## 6. 関数

```php
<?php
// 型宣言付き関数（引数と戻り値に型を書ける）
function calcTax(float $price, float $taxRate = 0.1): float {
    return $price * (1 + $taxRate);
}

echo calcTax(1000);       // → 1100（デフォルト 10%）
echo calcTax(1000, 0.08); // → 1080（8%）

// 複数の値を返す（連想配列で返す）
function calcStats(array $numbers): array {
    return [
        "total"   => array_sum($numbers),
        "average" => array_sum($numbers) / count($numbers),
        "max"     => max($numbers),
        "min"     => min($numbers),
    ];
}

$stats = calcStats([85, 72, 91, 68]);
echo "平均: " . $stats["average"];  // → 平均: 79

// アロー関数（PHP 7.4以降）
$double = fn($n) => $n * 2;
echo $double(5);  // → 10
```

## 7. フォームとデータの受け取り（$_POST）

```
┌──────────────────────┐     POST送信     ┌──────────────────────┐
│  form.html           │  ──────────────→  │  process.php         │
│  <form method="post"> │                  │  $_POST["namae"]     │
│  <input name="namae"> │                  │  処理してHTMLを返す   │
└──────────────────────┘                  └──────────────────────┘
```

### form.html

```html
<form action="process.php" method="post">
  <input type="text"  name="namae"   placeholder="名前">
  <input type="number" name="age"    placeholder="年齢">
  <textarea name="message"></textarea>
  <button type="submit">送信する</button>
</form>
```

### process.php

```php
<?php
$namae   = $_POST["namae"]   ?? "";
$age     = $_POST["age"]     ?? "";
$message = $_POST["message"] ?? "";

// !! 重要 !! XSS対策：htmlspecialchars でHTMLタグを無効化
$safeName    = htmlspecialchars($namae,   ENT_QUOTES, 'UTF-8');
$safeMessage = htmlspecialchars($message, ENT_QUOTES, 'UTF-8');

echo "<p>名前: {$safeName}</p>";
echo "<p>年齢: {$age}歳</p>";
echo "<p>メッセージ: {$safeMessage}</p>";
?>
```

## 8. XSS 対策

```php
<?php
// ❌ 危険な例（htmlspecialchars なし）
$input = "<script>alert('ハッキング！')</script>";
echo $input;
// → ブラウザが <script> を実行してしまう！

// ✅ 安全な例
echo htmlspecialchars($input, ENT_QUOTES, 'UTF-8');
// → &lt;script&gt;... とただの文字列として表示される
?>
```

> **ルール：** ユーザーの入力は必ず `htmlspecialchars()` を通してから表示する！

## 9. バリデーション（入力チェック）

```php
<?php
$errors = [];
$namae  = "";
$age    = "";

if ($_SERVER["REQUEST_METHOD"] === "POST") {
    $namae = trim($_POST["namae"] ?? "");
    $age   = trim($_POST["age"]   ?? "");

    if (empty($namae)) {
        $errors[] = "名前は必須です";
    } elseif (mb_strlen($namae) > 50) {
        $errors[] = "名前は50文字以内にしてください";
    }

    if (empty($age)) {
        $errors[] = "年齢は必須です";
    } elseif (!is_numeric($age) || $age < 0 || $age > 150) {
        $errors[] = "年齢は0〜150の数値を入力してください";
    }

    if (empty($errors)) {
        $safeName = htmlspecialchars($namae, ENT_QUOTES, 'UTF-8');
        echo "<p>こんにちは、{$safeName}さん（{$age}歳）！</p>";
    }
}

if (!empty($errors)) {
    echo "<ul style='color:red;'>";
    foreach ($errors as $error) {
        echo "<li>" . htmlspecialchars($error) . "</li>";
    }
    echo "</ul>";
}
?>
```

## まとめ

| 機能 | JavaScript | PHP |
| --- | --- | --- |
| 変数 | `let x = 1` | `$x = 1` |
| 配列の長さ | `arr.length` | `count($arr)` |
| 末尾に追加 | `arr.push(v)` | `$arr[] = v` |
| forEach | `arr.forEach(fn)` | `foreach ($arr as $v)` |
| オブジェクト | `{key: value}` | `["key" => value]` |
| else if | `else if` | `elseif` |
| アロー関数 | `(x) => x * 2` | `fn($x) => $x * 2` |
| フォームデータ受信 | `fetch` / `FormData` | `$_POST["name"]` |
| XSS 対策 | `textContent` を使う | `htmlspecialchars()` |
