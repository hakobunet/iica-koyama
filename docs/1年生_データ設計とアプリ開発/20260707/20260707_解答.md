## Part 1｜配列・連想配列

### 問題1-1 解答

```php
<?php
$scores = [85, 42, 91, 67, 78, 55, 93, 70, 61, 88];

// ① 合計点を出力する
$total = array_sum($scores);
echo "合計: {$total}<br>";

// ② 平均点を出力する（小数点第1位まで）
$avg = $total / count($scores);
echo "平均: " . number_format($avg, 1) . "<br>";

// ③ 最高点と最低点を出力する
echo "最高点: " . max($scores) . "<br>";
echo "最低点: " . min($scores) . "<br>";

// ④ 70点以上の合格者数を出力する
$passed = array_filter($scores, fn($s) => $s >= 70);
echo "合格者数: " . count($passed) . "<br>";
?>
```

### 問題1-2 解答

```php
<?php
$students = [
    ["id" => 1, "name" => "田中太郎", "class" => "A", "score" => 85],
    ["id" => 2, "name" => "鈴木花子", "class" => "B", "score" => 62],
    ["id" => 3, "name" => "山田次郎", "class" => "A", "score" => 91],
    ["id" => 4, "name" => "佐藤三郎", "class" => "B", "score" => 47],
    ["id" => 5, "name" => "伊藤美咲", "class" => "A", "score" => 78],
];

// ① 全生徒を「ID:1 田中太郎 (Aクラス) 85点」の形式で出力する
foreach ($students as $s) {
    echo "ID:{$s['id']} {$s['name']} ({$s['class']}クラス) {$s['score']}点<br>";
}

// ② クラスAの生徒だけを出力する
echo "<br>-- クラスA --<br>";
foreach ($students as $s) {
    if ($s["class"] === "A") {
        echo "{$s['name']}<br>";
    }
}

// ③ 80点以上の生徒だけを出力する
echo "<br>-- 80点以上 --<br>";
foreach ($students as $s) {
    if ($s["score"] >= 80) {
        echo "{$s['name']}<br>";
    }
}
?>
```

## Part 2｜条件分岐・関数

### 問題2-1 解答

```php
<?php
function getGrade(int $score): string {
    if ($score >= 90) {
        return "S";
    } elseif ($score >= 80) {
        return "A";
    } elseif ($score >= 70) {
        return "B";
    } elseif ($score >= 60) {
        return "C";
    } else {
        return "D";
    }
}

// 動作確認
$testScores = [95, 83, 72, 61, 45];
foreach ($testScores as $score) {
    echo "{$score}点 → " . getGrade($score) . "評価<br>";
}
// 期待する出力:
// 95点 → S評価
// 83点 → A評価
// 72点 → B評価
// 61点 → C評価
// 45点 → D評価
?>
```

<?php
function getAgeGroup(int $age): string {
    return match (true) {
        $age < 12 => "子ども",
        $age < 18 => "中高生",
        $age < 65 => "大人",

    };
}

echo getAgeGroup(15) . "<br>";  // → 中高生
echo getAgeGroup(30) . "<br>";  // → 大人
echo getAgeGroup(70) . "<br>";  // → シニア
?>

### 問題2-3 解答（発展）

```php
<?php
function addTax(array $products, float $taxRate = 0.1): array {
    return array_map(function ($p) use ($taxRate) {
        $p["price_with_tax"] = (int) round($p["price"] * (1 + $taxRate));
        return $p;
    }, $products);
}

$products = [
    ["name" => "ノートPC",   "price" => 98000],
    ["name" => "マウス",     "price" => 3200],
    ["name" => "キーボード", "price" => 8500],
];

$result = addTax($products);
foreach ($result as $p) {
    echo "{$p['name']}: {$p['price_with_tax']}円<br>";
}
// 期待:
// ノートPC: 107800円
// マウス: 3520円
// キーボード: 9350円
?>
```

## Part 3｜フォームと POST

### 問題3-1 解答（contact.php）

```php
<?php
// contact.php

$name    = "";
$email   = "";
$message = "";
$sent    = false;

if ($_SERVER["REQUEST_METHOD"] === "POST") {
    // ① データを取り出す（?? "" でnull対策）
    $name    = $_POST["name"] ?? "";
    $email   = $_POST["email"] ?? "";
    $message = $_POST["message"] ?? "";

    // ② htmlspecialchars を通す
    $name    = htmlspecialchars($name, ENT_QUOTES, "UTF-8");
    $email   = htmlspecialchars($email, ENT_QUOTES, "UTF-8");
    $message = htmlspecialchars($message, ENT_QUOTES, "UTF-8");

    $sent = true;
}
?>
<!DOCTYPE html>
<html lang="ja">
<head><meta charset="UTF-8"><title>問い合わせ</title></head>
<body>
<?php if ($sent): ?>
  <!-- 送信完了メッセージ -->
  <p><?= $name ?>さんからのお問い合わせを受け付けました</p>
<?php else: ?>
  <!-- フォーム -->
  <form method="post" action="">
    <label>名前：<input type="text" name="name"></label><br>
    <label>メールアドレス：<input type="email" name="email"></label><br>
    <label>メッセージ：<textarea name="message"></textarea></label><br>
    <button type="submit">送信</button>
  </form>
<?php endif; ?>
</body>
</html>
```

※ `<p>{$name}さんからのお問い合わせを受け付けました</p>` の部分は実際には
`<p><?= $name ?>さんからのお問い合わせを受け付けました</p>` のように `<?= ?>` タグで出力すること。

### 問題3-2 解答（register.php）

```php
<?php
$errors = [];
$name   = "";
$age    = "";

if ($_SERVER["REQUEST_METHOD"] === "POST") {
    $name = trim($_POST["name"] ?? "");
    $age  = trim($_POST["age"]  ?? "");

    // ① 名前のバリデーション
    if ($name === "") {
        $errors[] = "名前は必須です";
    } elseif (mb_strlen($name) > 50) {
        $errors[] = "名前は50文字以内で入力してください";
    }

    // ② 年齢のバリデーション
    if ($age === "") {
        $errors[] = "年齢は必須です";
    } elseif (!ctype_digit($age) || (int) $age < 0 || (int) $age > 120) {
        $errors[] = "年齢は0〜120の整数で入力してください";
    }

    // ③ エラーなしなら成功メッセージ
    $success = empty($errors);
}
?>
<!DOCTYPE html>
<html lang="ja">
<head><meta charset="UTF-8"><title>登録</title></head>
<body>
  <!-- ④ エラーがあれば赤字で表示 -->
  <?php if (!empty($errors)): ?>
    <ul style="color: red;">
      <?php foreach ($errors as $error): ?>
        <li><?= htmlspecialchars($error) ?></li>
      <?php endforeach; ?>
    </ul>
  <?php elseif (!empty($success)): ?>
    <p>登録が完了しました</p>
  <?php endif; ?>

  <form method="post" action="">
    <label>名前：<input type="text" name="name" value="<?= htmlspecialchars($name) ?>"></label><br>
    <label>年齢：<input type="number" name="age" value="<?= htmlspecialchars($age) ?>"></label><br>
    <button type="submit">登録</button>
  </form>
</body>
</html>
```
