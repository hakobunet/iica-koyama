## 問題3：1から10までの合計を計算する

`for` ループを使って 1〜10 の整数をすべて足した合計を出力してください。

**期待する出力:**
```
合計: 55
```

条件：
- `for` ループを使うこと
- 合計を格納する変数 `$total` を使うこと

```php
<?php
$total = 0;

// ここに書こう


echo "合計: " . $total;
?>
```
