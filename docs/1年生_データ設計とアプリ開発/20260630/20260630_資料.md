## 1. クライアントサイドとサーバーサイドの違い

```
クライアントサイド（前期で学んだもの）
  ブラウザで動く → HTML / CSS / JavaScript
  ユーザーの端末で実行される

サーバーサイド（今期から学ぶもの）
  サーバーで動く → PHP / Python / Ruby / Go など
  サーバーで実行されてからブラウザに結果が送られる
```

**なぜサーバーサイドが必要か？**

| やりたいこと | クライアントだけ | サーバーが必要 |
| --- | --- | --- |
| テキストを表示 | ✅ できる | — |
| データベースを読む | ❌ できない | ✅ 必要 |
| パスワードを安全に管理 | ❌ 危険 | ✅ 必要 |
| ログイン機能 | ❌ できない | ✅ 必要 |

## 2. PHP とは

**PHP = サーバーサイドのプログラミング言語。Webアプリ開発に特化。**

```
ブラウザ（あなた）          サーバー
    │                         │
    │── http://example.com ──→│
    │                         │ ← PHP が動く
    │                         │   データベースを見たり
    │                         │   計算をしたり
    │← HTML を返す ───────────│
    │
HTMLを表示する
```

**JavaScript と比べてみよう：**

| 項目 | JavaScript | PHP |
| --- | --- | --- |
| 動く場所 | ブラウザ | サーバー |
| 変数の書き方 | `let x = 5;` | `$x = 5;` |
| 文字列連結 | `"Hello" + "World"` | `"Hello" . "World"` |
| 配列 | `let arr = [1, 2, 3]` | `$arr = [1, 2, 3]` |
| 出力 | `console.log()` | `echo` |

## 3. 環境構築：PHP のインストール

### Mac の場合（Homebrew を使用）

Homebrew が未インストールの場合は [brew.sh](https://brew.sh) の手順で先にインストールしてください。

```bash
brew install php
```

### Windows の場合

1. [windows.php.net/download](https://windows.php.net/download/) にアクセス
2. 「VS16 x64 Non Thread Safe」の ZIP をダウンロード
3. `C:\php` に展開
4. 環境変数 PATH に `C:\php` を追加

### 動作確認

```bash
php --version
# → PHP 8.x.x ... と表示されればOK
```

## 4. はじめての PHP ファイル

### 作業フォルダを作る(どこでも構いません)

```
Mac:     ~/study/
Windows: C:\Users\ユーザー名\study\
```

### `index.php` を作成

```php
<?php
// PHP のコードは <?php から始まる
// JS の console.log に相当するのが echo

echo "こんにちは PHP！";

// 変数は $ から始まる（JavaScript との大きな違い）
$namae = "田中太郎";
$nenrei = 20;

// 文字列の中に変数を埋め込む（ダブルクォートのみ）
echo "名前: $namae";
echo "年齢: {$nenrei}歳";  // {} で囲むと確実

// JS のテンプレートリテラルに相当
// JS: `名前: ${namae}`
// PHP: "名前: {$namae}"
?>
```

**ローカルサーバーを起動してブラウザで確認：**

```bash
# study フォルダに移動してから実行
cd ~/study
php -S localhost:8000
```

`http://localhost:8000/index.php` にアクセス

## 5. PHP の基本構文（JS と比較）

```php
<?php
// ========== 変数 ==========
$name = "Alice";      // JS の let に相当（$ が必要）
$age = 20;
$pi = 3.14;
$isStudent = true;

// ========== 出力 ==========
echo "Hello";          // 改行なしで出力
echo "World\n";        // \n で改行
echo $name . "\n";     // . で文字列連結（JS の + に相当）

// PHP タグと HTML を混在できる（PHP の特徴）
?>
<h1>こんにちは、<?= $name ?>さん！</h1>
<!-- <?= は echo の省略形 -->
<?php

// ========== 条件分岐（JS と同じ感覚） ==========
if ($age >= 18) {
    echo "成人です\n";
} else {
    echo "未成年です\n";
}

// ========== ループ ==========
for ($i = 1; $i <= 5; $i++) {   // $i は JS の i と同じ
    echo $i . "回目\n";
}

// ========== 配列 ==========
$fruits = ["りんご", "みかん", "ぶどう"];
echo $fruits[0] . "\n";  // → りんご

// 配列のループ
foreach ($fruits as $fruit) {  // JS の forEach に相当
    echo $fruit . "\n";
}
?>
```

| 概念 | JavaScript | PHP |
| --- | --- | --- |
| 変数 | `let x` | `$x` |

| 配列ループ | `forEach` | `foreach` |
| 動く場所 | ブラウザ | サーバー |
