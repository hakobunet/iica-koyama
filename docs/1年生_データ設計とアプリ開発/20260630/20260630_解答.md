## 問題2 解答

```php
<?php
$fruits = ["りんご", "みかん", "ぶどう", "バナナ"];

echo "<ul>";
foreach ($fruits as $fruit) {
    echo "<li>{$fruit}</li>";
}
echo "</ul>";
?>
```

## 問題3 解答

```php
<?php
$total = 0;

for ($i = 1; $i <= 10; $i++) {
    $total += $i;
}

echo "合計: " . $total;  // → 合計: 55
?>
```

## 問題4 解答（発展）

```php
<?php
for ($dan = 2; $dan <= 5; $dan++) {
    for ($i = 1; $i <= 9; $i++) {
        echo "{$dan}×{$i}=" . ($dan * $i) . "  ";
    }
    echo "<br>";
}
?>
```
